## Descripción

El presente repositorio contiene un control de versiones, para la bóveda de conocimiento del curso, donde se tendrán los archivos relacionados al obsidian. Esto con el fin de tener una referencia y respaldo para los apuntes del curso.

En la carpeta "CE1116_Notas" se encuentran las notas realizadas a lo largo del curso, ya sea notas de las clases o de asignaciones como tareas que deje el profesor. En el caso de la carpeta "Referencia", se encuentra los archivos de inicialización de la bóveda compartidos por el profesor, para así tenerlos de referencia en caso de que se necesiten para futuras tareas o para la organización de la bóveda.
