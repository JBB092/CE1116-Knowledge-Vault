# Bóveda de Conocimiento

## Descripción de la Bóveda

Esta bóveda de conocimiento fue creada como un repositorio personal de conceptos fundamentales relacionados con:

- Ingeniería de Requerimientos  
- Filosofía Ágil  
- Framework Scrum  
- Planificación y ejecución de proyectos  
- Estimación y métricas  

El objetivo principal de esta bóveda es servir como herramienta de estudio, consulta rápida y referencia académica durante el curso.

---

## Organización de la Bóveda

La bóveda está organizada por temáticas principales:

- **Ingeniería de Requerimientos**
- **Filosofía Ágil y Framework Scrum**
- **Estimación y Métricas en Scrum**
- **Gestión de Partes Interesadas**
- **Planificación y Ejecución (Historias de Usuario)**

Cada carpeta contiene las notas correspondientes a los conceptos solicitados en la asignación y otros conceptos complementarios.

---

## Uso de la Bóveda

Las notas están interconectadas mediante enlaces internos ([[ ]]) para facilitar la navegación y la construcción de una red de conocimiento.

Se recomienda utilizar:

- El panel de grafo de Obsidian para visualizar relaciones.
- Las carpetas temáticas para explorar conceptos por área.
- Las secciones de ejemplos para reforzar el aprendizaje práctico.
