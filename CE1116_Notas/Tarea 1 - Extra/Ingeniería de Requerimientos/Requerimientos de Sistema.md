## Definición
Los Requerimientos de Sistema son especificaciones detalladas que describen qué debe hacer el sistema para satisfacer las metas del usuario y cumplir los objetivos del negocio, incluyendo el mayor nivel de detalle técnico.

## Ejemplo
“El sistema debe permitir que el usuario inicie sesión utilizando correo electrónico y contraseña.”

## Notas
- Son más detallados que los de negocio.
- Incluyen aspectos funcionales y no funcionales.
- Guían directamente el desarrollo del sistema.
- Son la base para las pruebas.
